URL'ы
=====

Все URL в CMS "Proxima CMS" должны удовлетворять следующим параметрам:
  - быть не длиннее 128 символов;
  - содержать только [[:alnum:]], "-", "_", ".";

Урлы структурных элементов должны
  - не содержать ".";

Урлы контентных элементов должны
  - содержать одну и только одну ".";


Попытка структурировать *.ini
=============================

Форматы данных содержат ряд параметров:
title           = СТРОКА                            |
base            = [struct|content|subcontent|user]  |
parent          = [formatname]                      |
childs          = service, phone, allowed           |
order           = [fieldname]                       |
pathname        = [fieldname]                       |
nullrole        = 16384                             |
referenced      = [formatname]                      |


f:id            = pk
f:parent        = fk
f:title         = string
f:pathname      = string
f:description   = string
f:status        = boolean
