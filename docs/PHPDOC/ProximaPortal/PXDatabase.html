AddContentObject (line 623)

Добавляет объект в БД.

Перед добавлением объект преобразуется к необходимому storage type. Перед добавлением объекта вызываются триггеры. Если хотя бы один OnBeforeAddObject триггер вернёт false, добавление не произойдёт. После успешного добавления объекта вызываются триггеры OnAddObject. Метод использует PXDatabase::InsertObject для добавления объектов.

  • see: InsertObject
